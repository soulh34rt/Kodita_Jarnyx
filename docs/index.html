Cómo funciona

Kodita Jarnyx transforma cada letra según su posición en la palabra y la longitud de esa palabra, usando desplazamiento circular sobre un alfabeto multilingüe de más de 500 caracteres.

v2.0: Nuevo algoritmo robusto con desplazamiento circular. Soporte multilenguaje.

Codificar

  1. Longitud de la palabra (semilla)
  2. Posición de cada letra en el alfabeto
  3. (posición + semilla + índice) mod tamaño

Decodificar

  1. Longitud de la palabra cifrada (semilla)
  2. Posición de cada letra en el alfabeto
  3. (posición − semilla − índice) mod tamaño

Ejemplo: "Hola こんにちは"

Codificación:

'H' (33) → (33 + 4 + 1) = 38 → 'L'
'o' (44) → (44 + 4 + 2) = 50 → 'd'
'l' (37) → (37 + 4 + 3) = 44 → 'o'
'a' (26) → (26 + 4 + 4) = 34 → 'J'
Espacio se preserva
'こ' (100) → (100 + 5 + 1) = 106 → 'ぶ'
'ん' (101) → (101 + 5 + 2) = 108 → 'り'
'に' (102) → (102 + 5 + 3) = 110 → 'ど'
'ち' (103) → (103 + 5 + 4) = 112 → 'ふ'
'は' (104) → (104 + 5 + 5) = 114 → 'む'

Resultado: "LdoJ ぶりどふむ"

Alfabeto: latín, acentos, japonés, chino, coreano, cirílico
